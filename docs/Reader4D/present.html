Reader4D.present

Utilities for generating presentation-ready outputs from Virtual4D results, including:

  • Applying a virtual detector configuration specified in an Excel sheet.
  • Building image sequences (GIF/MP4) from exported PNGs.
  • Optionally overlaying per-frame text annotations sourced from Excel tables.

This module is designed to work with Virtual4D virtual detector objects (e.g. Virtual4D.virtDets.Annular) and the output images created by the Virtual4D pipeline.

Notes

  • This module assumes a Windows environment for font discovery by default. If Times New Roman is not available, Pillow's default font is used.
  • For image sequence creation, all frames are resized to a uniform size (that of the first frame) to satisfy GIF/MP4 encoders.

 31class Creator:
 32    """
 33    Convenience helper to (a) build or reuse a Virtual4D detector and
 34    (b) generate derived outputs (export series, movies) for presentation.
 35
 36    Typical workflow
 37    ----------------
 38    1) Instantiate :class:`Creator` with a representative diffractogram
 39       ``pattern``, plus ``packets`` and ``descriptors``.
 40    2) Either provide an existing detector instance via ``virtdet`` or let
 41       the class construct an annular detector automatically.
 42    3) Use :meth:`Excel` to apply detector ranges from a spreadsheet.
 43    4) Use :meth:`ImageSeries2` to build an annotated GIF/MP4 from the
 44       exported images.
 45
 46    Parameters
 47    ----------
 48    pattern : numpy.ndarray
 49        Representative diffractogram used by Virtual4D for detector
 50        visualization and initialization.
 51
 52    packets, descriptors : numpy.ndarray
 53        Sparse data structures needed by Virtual4D detector reconstruction.
 54
 55    values_role : {"count", "itot"}
 56        Which packet field Virtual4D should interpret as signal.
 57
 58    out_dir : str or os.PathLike
 59        Output directory used for saving generated images/movies.
 60
 61    virtdet : object, optional
 62        Pre-constructed virtual detector instance. If provided, the Creator
 63        will reuse it. If None, an annular detector is created.
 64
 65    scan_dims : tuple[int, int], default (1024, 768)
 66        Scan dimensions as (width, height), passed through to Virtual4D.
 67
 68    rLower, rUpper : float
 69        Default inner/outer detector radii passed to Virtual4D annular
 70        detector initialization. Interpretation depends on Virtual4D.
 71
 72    sigma : float or None
 73        Optional Gaussian blur parameter for the annular detector response.
 74
 75    show : bool, default True
 76        If True, Virtual4D may display intermediate visualizations.
 77
 78    cmap : str, default "gray"
 79        Colormap used by Virtual4D plotting utilities.
 80
 81    verbose : int, default 1
 82        Verbosity for downstream tools.
 83
 84    save_im : bool, default True
 85        Whether Virtual4D should save images when applying detector filters.
 86
 87    name_im : str or None
 88        Optional base name used by some Virtual4D saving functions.
 89
 90    Attributes
 91    ----------
 92    VIRTDET : object
 93        Virtual detector instance (either supplied or created).
 94    """

 97                 pattern,
 98                 packets,
 99                 descriptors,
100                 values_role,
101                 out_dir,
102                 virtdet=None,
103                 scan_dims = (1024, 768),
104                 rLower = 0.01,
105                 rUpper = 0.20,
106                 sigma = None,
107                 show = True,
108                 cmap = "gray",
109                 verbose = 1,
110                 save_im = True,
111                 name_im = None,
112                 ):
113        """
114        Apply a sequence of detector inner/outer radii specified in an Excel 
115        file.
116
117        The spreadsheet is expected to contain one row per detector 
118        configuration. Each row is mapped to a call of::
119
120            self.VIRTDET.filter_centered_ROI(lower=..., upper=..., ...)
121
122        Parameters
123        ----------
124        xlsx_path : str or os.PathLike
125            Path to an Excel workbook.
126
127        sheet_name : str or int or None, optional
128            Sheet identifier passed to :func:`pandas.read_excel`.
129            If None, pandas uses the first sheet.
130
131        columns : sequence[str], default ("idx","inner","outer")
132            Column names used to extract:
133            - index/id label (used for naming)
134            - inner radius
135            - outer radius
136
137        Notes
138        -----
139        Naming convention uses a counter within each `idx` group. The counter
140        resets when `idx` changes. The first file for a new `idx` will be
141        numbered `00` by default in this implementation.
142        """
143        
144        self.pattern = pattern
145        self.packets = packets
146        self.descriptors = descriptors
147        self.values_role = values_role
148
149        if isinstance(out_dir, tuple):
150            if len(out_dir) == 1:
151                out_dir = out_dir[0]
152            else:
153                raise TypeError(f"out_dir should be str, not tuple: {out_dir}")
154        self.OUT_DIR = str(out_dir)
155
156        self.SCAN_DIMS = scan_dims
157        self.LOWER = rLower
158        self.UPPER = rUpper
159        self.SIGMA = sigma
160        self.SHOW = show
161        self.CMAP = cmap
162        self.VERBOSE = verbose
163        self.SAVE_IM = save_im
164        self.NAME_IM = name_im
165        
166        
167        # Initialize virtual detector
168        if virtdet is not None:
169            self.VIRTDET=virtdet
170        else:
171            self.VIRTDET = v4dVDet.Annular(
172                pattern,
173                self.packets,
174                self.descriptors,
175                self.values_role,
176                self.OUT_DIR,
177                scan_dims=self.SCAN_DIMS,
178                mode=2,
179                rLower=self.LOWER,
180                rUpper=self.UPPER,
181                sigma=self.SIGMA,
182                show=self.SHOW,
183                cmap=self.CMAP,
184                verbose=self.VERBOSE,
185                save_im=False,
186                name_im=self.NAME_IM
187                )

189    def Excel(self,
190              xlsx_path,
191              sheet_name = None,
192              columns = ['idx', 'inner', 'outer']):
193        """
194        Apply a sequence of detector inner/outer radii specified in an Excel 
195        file.
196
197        The spreadsheet is expected to contain one row per detector 
198        configuration. Each row is mapped to a call of::
199
200            self.VIRTDET.filter_centered_ROI(lower=..., upper=..., ...)
201
202        Parameters
203        ----------
204        xlsx_path : str or os.PathLike
205            Path to an Excel workbook.
206
207        sheet_name : str or int or None, optional
208            Sheet identifier passed to :func:`pandas.read_excel`.
209            If None, pandas uses the first sheet.
210
211        columns : sequence[str], default ("idx","inner","outer")
212            Column names used to extract:
213            - index/id label (used for naming)
214            - inner radius
215            - outer radius
216
217        Notes
218        -----
219        Naming convention uses a counter within each `idx` group. The counter
220        resets when `idx` changes. The first file for a new `idx` will be
221        numbered `00` by default in this implementation.
222        """
223        
224        # Load EXCEL data
225        df = pd.read_excel(xlsx_path, 
226                           sheet_name)
227
228        # Select the desired columns
229        subset = df.loc[:, columns]
230        counter = 0
231        index = subset[columns[0]][0]
232        
233
234            
235            
236        for i in range(0,len(subset)):
237            s = subset[columns[0]][i]
238            LOWER = subset[columns[1]][i]
239            UPPER = subset[columns[2]][i]
240            
241            if s == index:
242                counter = counter+1
243            else:
244                counter = 0
245                index = s
246        
247            if counter<10:
248                prefix = f"{s}0{counter}_"
249            else:
250                prefix = f"{s}{counter}_"
251        
252            FILENAME = prefix+f"lower{LOWER}_upper{UPPER}.png"
253            
254            # Application of virtual detectors
255            self.VIRTDET.filter_centered_ROI(
256                lower=LOWER,
257                upper=UPPER,
258                name_im=FILENAME,
259                
260                sigma=self.SIGMA)
261            
262        return

265    def ImageSeries2(self, path, filename,
266                xlsx=None, sheet=None, cols=None, ids=None,
267                *,
268                fps=10,
269                base_font_px=40,
270                pad_x=20, pad_y=14,
271                vis_lo_pct=1.0, vis_hi_pct=99.5,
272                clip_pct=None):
273        """
274        Build an animated GIF/MP4 from PNG images, overlaying a text line
275        derived from an Excel table.
276
277        This is the recommended variant: it supports more robust mapping 
278        between Excel rows and filenames, filtering by `idx`, and per-frame 
279        robust contrast scaling for presentation.
280
281        Parameters
282        ----------
283        path : str or os.PathLike
284            Directory containing PNG frames.
285
286        filename : str
287            Output filename. The extension controls encoding behavior.
288            Common choices: ``.gif`` or ``.mp4``.
289
290        xlsx : str or os.PathLike, optional
291            Excel workbook used to generate the per-frame text overlays.
292
293        sheet : str or int, optional
294            Excel sheet identifier.
295
296        cols : sequence[str] of length 2, optional
297            Column names used to build the overlay text, e.g.
298            ``["inner [mrad]", "outer [mrad]"]``.
299            Required when ``xlsx`` is provided.
300
301        ids : str or iterable[str], optional
302            Optional filter for frames, using the leading letter in the PNG
303            basename (e.g., "A", "B"). If provided and the sheet contains an
304            ``idx`` column, the table is also filtered accordingly.
305
306        fps : int, default 10
307            Output frames-per-second for the encoded animation.
308
309        base_font_px : int, default 40
310            Starting font size. The function will shrink this if the widest
311            annotation line does not fit within the frame width.
312
313        pad_x, pad_y : int
314            Horizontal/vertical padding (in pixels) for the text strip.
315
316        vis_lo_pct, vis_hi_pct : float
317            Robust per-channel percentiles used for display scaling of PNGs.
318            This affects only the rendered movie frames (not source images).
319
320        clip_pct : float or None
321            Optional high-percentile cap applied before scaling (useful for
322            suppressing occasional hot pixels in display-only output).
323
324        Notes
325        -----
326        - All frames are resized to match the first frame dimensions.
327        - For sparse-to-dense reconstruction, do that upstream and export PNGs.
328        """
329   
330        # ---- gather files
331        files_all = sorted(glob.glob(os.path.join(path, "*.png")))
332        if not files_all:
333            raise FileNotFoundError(f"No .png files in {path}")
334    
335        # normalize ids input -> set[str] (uppercase)
336        if ids is None:
337            want_ids = None
338        elif isinstance(ids, str):
339            want_ids = {ids.upper()}
340        else:
341            want_ids = {str(x).upper() for x in ids}
342    
343        # helper: get the letter id from a filename (first AZ at start)
344        def file_idx_letter(f):
345            """Return the leading AZ letter used as a frame/group ID, or None.
346            """
347            
348            m = re.match(r"([A-Za-z])", os.path.basename(f))
349            return m.group(1).upper() if m else None
350    
351        # ---- load Excel (optional) and filter rows by ids
352        texts_by_stem = None
353        ordered_stems = None
354        df = None
355        if xlsx is not None:
356            df = pd.read_excel(xlsx, sheet_name=sheet)
357    
358            # Filter DF by ids if provided and 'idx' exists
359            if want_ids is not None and "idx" in df.columns:
360                df = df[df["idx"].astype(str).str.upper().isin(want_ids)]
361    
362            # columns to read text from
363            if cols is None or len(cols) != 2:
364                raise ValueError(
365                    "`cols` must be a list of two column names, e.g. ['inner [mrad]','outer [mrad]'].")
366    
367            # If DF has a filename-like column, build mapping {stem: text} and keep table order
368            fname_col = next(
369                (c for c in ("file","filename","name","stem","basename") if \
370                 c in df.columns), None)
371            if fname_col is not None:
372                # normalize stems from table
373                stem_series = df[fname_col].astype(str).str.replace(
374                    r"\.png$", "", regex=True).str.strip()
375                ordered_stems = stem_series.tolist()
376    
377                texts_by_stem = {}
378                for _, row in df.iterrows():
379                    stem = str(row[fname_col]).strip()
380                    stem = re.sub(r"\.png$", "", stem, flags=re.IGNORECASE)
381                    l = float(row[cols[0]])
382                    u = float(row[cols[1]])
383                    texts_by_stem[stem] = f"Range: {l:.0f}-{u:.0f} mrad"
384    
385        # ---- select files to process + text per file
386        proc_files = []
387        proc_texts = []
388    
389        if texts_by_stem is not None:
390            # Use stems from table (already filtered by ids if applicable)
391            # Match to actual files; skip stems that don't exist on disk.
392            files_by_stem = {os.path.splitext(os.path.basename(f))[0]: f for f in files_all}
393            for stem in ordered_stems:
394                f = files_by_stem.get(stem)
395                if f is None:
396                    # try a loose match if the table stem is a prefix
397                    match = next((ff for st, ff in files_by_stem.items() if st.startswith(stem)), None)
398                    if match is None:
399                        continue
400                    f = match
401                # If ids requested, enforce by filename too
402                if want_ids is not None and file_idx_letter(f) not in want_ids:
403                    continue
404                proc_files.append(f)
405                proc_texts.append(texts_by_stem.get(os.path.splitext(os.path.basename(f))[0], ""))
406        else:
407            # No filename column in Excel.
408            # Strategy: filter files by ids (if any), then assign texts per-id 
409            # in row order.
410            files = files_all
411            if want_ids is not None:
412                files = [f for f in files_all if file_idx_letter(f) in want_ids]
413    
414            if df is None:
415                # No Excel at all -> no text
416                proc_files = files
417                proc_texts = [""] * len(files)
418            else:
419                # Build per-id queues of text in DF order
420                if "idx" in df.columns:
421                    df["_IDX_UP"] = df["idx"].astype(str).str.upper()
422                else:
423                    # If theres no 'idx' column, we cant align by id; just 
424                    # emit in DF order vs files order length
425                    df["_IDX_UP"] = ""
426    
427                # queues: { 'A': [text1, text2, ...], ... }
428                queues = {}
429                for _, row in df.iterrows():
430                    l = float(row[cols[0]])
431                    u = float(row[cols[1]])
432                    txt = f"Range: {l:.0f}-{u:.0f} mrad"
433                    key = row["_IDX_UP"]
434                    queues.setdefault(key, []).append(txt)
435    
436                for f in files:
437                    key = file_idx_letter(f) or ""
438                    txt = queues.get(key, [""]).pop(0) if queues.get(key) else ""
439                    proc_files.append(f)
440                    proc_texts.append(txt)
441    
442        if not proc_files:
443            raise RuntimeError(
444                "No frames matched the requested `ids`/Excel mapping."
445                )
446    
447        # ---- robust display scaling
448        def robust_scale(img_bgr, lo=1.0, hi=99.5, cap_pct=None):
449            """ Apply robust percentile-based intensity scaling for display.
450            """
451            arr = img_bgr.astype(np.float32)
452            if cap_pct is not None:
453                for ch in range(arr.shape[2]):
454                    hi_val = np.percentile(arr[..., ch], cap_pct)
455                    if np.isfinite(hi_val):
456                        arr[..., ch] = np.minimum(arr[..., ch], hi_val)
457            out = np.empty_like(arr, dtype=np.uint8)
458            for ch in range(arr.shape[2]):
459                lo_v = np.percentile(arr[..., ch], lo)
460                hi_v = np.percentile(arr[..., ch], hi)
461                if hi_v <= lo_v:
462                    out[..., ch] = 0
463                else:
464                    tmp = np.clip((arr[..., ch] - lo_v) / (hi_v - lo_v), 0, 1)
465                    out[..., ch] = (tmp * 255).astype(np.uint8)
466            return out
467    
468        # ---- font helpers
469        font_paths = [
470            r"C:\Windows\Fonts\times.ttf",
471            r"C:\Windows\Fonts\timesbd.ttf",
472            "times.ttf",
473        ]
474        def make_font(sz):
475            """Return a TrueType font of size `sz`, falling back to Pillow 
476            default."""
477
478            for p in font_paths:
479                try:
480                    return ImageFont.truetype(p, sz)
481                except OSError:
482                    continue
483            return ImageFont.load_default()
484    
485        def text_size(text, font):
486            """Return (width, height) of `text` rendered with `font` in pixels.
487            """
488
489            probe = Image.new("RGB", (1, 1))
490            draw = ImageDraw.Draw(probe)
491            try:
492                l, t, r, b = draw.textbbox((0, 0), text, font=font)
493                return (r - l), (b - t)
494            except AttributeError:
495                return draw.textsize(text, font=font)
496    
497        # ---- frame geometry + one font size that fits the widest line
498        base = cv2.imread(proc_files[0])
499        if base is None:
500            raise RuntimeError(f"Cannot read first image: {proc_files[0]}")
501        h0, w0 = base.shape[:2]
502    
503        widest = max(proc_texts, key=len, default="")
504        font = make_font(base_font_px)
505        target_width = w0 - 2 * pad_x
506        tw, th = text_size(widest, font) if widest else (0, base_font_px)
507        if widest and tw > target_width:
508            new_sz = max(10, int(base_font_px * target_width / max(tw, 1)))
509            font = make_font(new_sz)
510            tw, th = text_size(widest, font)
511            while widest and tw > target_width and new_sz > 10:
512                new_sz -= 1
513                font = make_font(new_sz)
514                tw, th = text_size(widest, font)
515    
516        strip_h = th + 2 * pad_y
517        canvas_h, canvas_w = h0 + strip_h, w0
518    
519        # ---- build frames
520        frames = []
521        for fpath, text in zip(proc_files, proc_texts):
522            img = cv2.imread(fpath)
523            if img is None:
524                continue
525            h, w = img.shape[:2]
526            if (h, w) != (h0, w0):
527                img = cv2.resize(img, (w0, h0), interpolation=cv2.INTER_AREA)
528    
529            img_disp = robust_scale(img, lo=vis_lo_pct, hi=vis_hi_pct, cap_pct=clip_pct)
530    
531            canvas = np.full((canvas_h, canvas_w, 3), 255, np.uint8)
532            canvas[:h0, :w0] = img_disp
533    
534            pil_img = Image.fromarray(cv2.cvtColor(canvas, cv2.COLOR_BGR2RGB))
535            draw = ImageDraw.Draw(pil_img)
536            tw_i, th_i = text_size(text, font)
537            x = max(pad_x, (canvas_w - tw_i) // 2)
538            y = h0 + (strip_h - th_i) // 2
539            draw.text((x, y), text, font=font, fill=(0, 0, 0))
540            canvas = cv2.cvtColor(np.array(pil_img), cv2.COLOR_RGB2BGR)
541    
542            frames.append(cv2.cvtColor(canvas, cv2.COLOR_BGR2RGB))
543    
544        print(f"[INFO] Frames in movie: {len(frames)}")
545    
546        # ---- save
547        out_path = os.path.join(path, filename)
548        ext = os.path.splitext(filename)[1].lower()
549        if ext == ".gif":
550            imageio.mimsave(out_path, frames, fps=2)
551        elif ext in (".mp4", ".mov", ".mkv"):
552            try:
553                imageio.mimsave(out_path, frames, fps, codec="libx264", quality=8)
554            except Exception:
555                imageio.mimsave(out_path, frames, fps=2)
556        else:
557            imageio.mimsave(out_path, frames, fps=2)
558    
559        print(f"[INFO] Movie saved to: {out_path}")

346            """Return the leading AZ letter used as a frame/group ID, or None.
347            """

350            return m.group(1).upper() if m else None

362    
363            # columns to read text from

423                else:
424                    # If theres no 'idx' column, we cant align by id; just 
425                    # emit in DF order vs files order length

427    
428                # queues: { 'A': [text1, text2, ...], ... }
429                queues = {}
430                for _, row in df.iterrows():
431                    l = float(row[cols[0]])
432                    u = float(row[cols[1]])
433                    txt = f"Range: {l:.0f}-{u:.0f} mrad"
434                    key = row["_IDX_UP"]
435                    queues.setdefault(key, []).append(txt)
436    
437                for f in files:
438                    key = file_idx_letter(f) or ""
439                    txt = queues.get(key, [""]).pop(0) if queues.get(key) else ""
440                    proc_files.append(f)
441                    proc_texts.append(txt)
442

445                "No frames matched the requested `ids`/Excel mapping."
446                )

448        # ---- robust display scaling
449        def robust_scale(img_bgr, lo=1.0, hi=99.5, cap_pct=None):
450            """ Apply robust percentile-based intensity scaling for display.
451            """
452            arr = img_bgr.astype(np.float32)
453            if cap_pct is not None:
454                for ch in range(arr.shape[2]):
455                    hi_val = np.percentile(arr[..., ch], cap_pct)
456                    if np.isfinite(hi_val):
457                        arr[..., ch] = np.minimum(arr[..., ch], hi_val)
458            out = np.empty_like(arr, dtype=np.uint8)
459            for ch in range(arr.shape[2]):
460                lo_v = np.percentile(arr[..., ch], lo)
461                hi_v = np.percentile(arr[..., ch], hi)
462                if hi_v <= lo_v:
463                    out[..., ch] = 0
464                else:
465                    tmp = np.clip((arr[..., ch] - lo_v) / (hi_v - lo_v), 0, 1)
466                    out[..., ch] = (tmp * 255).astype(np.uint8)
467            return out
468    
469        # ---- font helpers
470        font_paths = [
471            r"C:\Windows\Fonts\times.ttf",
472            r"C:\Windows\Fonts\timesbd.ttf",
473            "times.ttf",
474        ]

476            """Return a TrueType font of size `sz`, falling back to Pillow 
477            default."""
478
479            for p in font_paths:

482                except OSError:

486        def text_size(text, font):
487            """Return (width, height) of `text` rendered with `font` in pixels.
488            """
489
490            probe = Image.new("RGB", (1, 1))
491            draw = ImageDraw.Draw(probe)

494                return (r - l), (b - t)
495            except AttributeError:
496                return draw.textsize(text, font=font)
497    
498        # ---- frame geometry + one font size that fits the widest line
499        base = cv2.imread(proc_files[0])
500        if base is None:
501            raise RuntimeError(f"Cannot read first image: {proc_files[0]}")
502        h0, w0 = base.shape[:2]

508        if widest and tw > target_width:

510            font = make_font(new_sz)
511            tw, th = text_size(widest, font)
512            while widest and tw > target_width and new_sz > 10:
513                new_sz -= 1

515                tw, th = text_size(widest, font)
516

518        canvas_h, canvas_w = h0 + strip_h, w0
519    
520        # ---- build frames

527            if (h, w) != (h0, w0):

531    
532            canvas = np.full((canvas_h, canvas_w, 3), 255, np.uint8)

534    
535            pil_img = Image.fromarray(cv2.cvtColor(canvas, cv2.COLOR_BGR2RGB))
536            draw = ImageDraw.Draw(pil_img)

540            draw.text((x, y), text, font=font, fill=(0, 0, 0))
541            canvas = cv2.cvtColor(np.array(pil_img), cv2.COLOR_RGB2BGR)
542    
543            frames.append(cv2.cvtColor(canvas, cv2.COLOR_BGR2RGB))
544

550        if ext == ".gif":

555            except Exception:

Convenience helper to (a) build or reuse a Virtual4D detector and (b) generate derived outputs (export series, movies) for presentation.

Typical workflow

1) Instantiate Creator with a representative diffractogram pattern, plus packets and descriptors. 2) Either provide an existing detector instance via virtdet or let the class construct an annular detector automatically. 3) Use Excel() to apply detector ranges from a spreadsheet. 4) Use ImageSeries2() to build an annotated GIF/MP4 from the exported images.
