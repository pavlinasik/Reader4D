Reader4D.detectors

This module provides the Reader4D.detectors loaders, high-level convenience wrappers for working with 4D-STEM datasets.

The primary supported input format is a CSR triplet written by the acquisition pipeline:

  • indptr.raw : CSR row pointer (length = n_frames + 1)
  • indices.raw : detector addresses for nonzero events (length = nnz)
  • values.raw : per-event values (counts or iToT; length = nnz)
  • *.toml : sidecar metadata describing shapes, dtypes, acquisition parameters

After loading, the data are exposed in a downstream-friendly representation:

  • packets : structured array with dtype ~Reader4D.dtypes.TP3_ACQ_DATA_PACKET_DTYPE and fields ('address', 'count', 'itot')
  • descriptors : structured array with dtype ~Reader4D.dtypes.TP3_BIN_DESCRIPTOR_DTYPE and fields ('offset', 'packet_count')
  • header : a flat metadata/statistics dict derived from the TOML and from basic CSR statistics

The loader supports optional visualization and micrograph reconstruction (count and iToT), and can optionally export derived outputs to disk via Reader4D.convertor and Reader4D.visualizer.

Quick start

Load a CSR dataset (with TOML sidecar) and preview diffractograms::

import Reader4D.detectors as r4dDet

detector = r4dDet.Timepix3(
    in_dir=r"C:\path\to\csr_dataset",
    values_role="count",
    show=True,
    micrographs=True,
    progress=True,
    verbose=1,
)

packets = detector.packets
descriptors = detector.descriptors
header = detector.header

Reconstruct a single diffractogram for a scan position::

img, total = detector.get_diffractogram(
    packets=detector.packets,
    descriptors=detector.descriptors,
    pattern_index=0,
    scan_dims=detector.SCAN_DIMS,
    detector_dims=detector.DET_DIMS,
)

Notes

  • This module assumes that packets['address'] is a linear index into the detector plane (0..W*H-1). If your acquisition encodes addresses differently, you must decode the address prior to accumulation.
  • The CSR arrays are memory-mapped for efficient access and may be prefetched into the OS cache to reduce random I/O overhead for interactive exploration.

 96class Timepix3:
 97    """
 98    High-level loader for Timepix3 4D-STEM datasets stored as a CSR triplet.
 99
100    The loader expects a directory containing a CSR triplet (``indptr.raw``,
101    ``indices.raw``, ``values.raw``) and a TOML sidecar (``*.toml``) describing
102    dtypes and shapes. Data are exposed in a downstream-friendly form as
103    ``packets`` and ``descriptors`` structured arrays, plus an optional
104    normalized ``header`` dictionary.
105
106    Attributes
107    ----------
108    in_dir : str or os.PathLike
109        Input directory containing CSR files and a TOML sidecar.
110
111    out_dir : str or os.PathLike
112        Output directory used for derived artifacts (plots, exports). If not
113        provided, defaults to ``in_dir``.
114
115    indptr_file : str
116        Filename of the CSR row-pointer array. Typically ``"indptr.raw"``.
117        The row-pointer array has length ``n_frames + 1`` and must be
118        non-decreasing.
119
120    indices_file : str
121        Filename of the CSR indices array. Typically ``"indices.raw"``.
122        Holds detector addresses for each packet/event.
123
124    values_file : str
125        Filename of the CSR values array. Typically ``"values.raw"``.
126        Holds per-packet values interpreted as either counts or iToT.
127
128    values_role : {"count", "itot"}
129        How ``values_file`` should be mapped into the packet structure.
130        If ``"count"``, values populate ``packets["count"]`` and
131        ``packets["itot"]`` is zeroed. If ``"itot"``, the inverse mapping
132        is used.
133
134    scan_dims : tuple[int, int] or None
135        Scan grid dimensions stored as ``(width, height)``. If None, inferred
136        from the TOML ``nav_shape`` (which is commonly stored as ``(H, W)``).
137
138    det_dims : tuple[int, int] or None
139        Detector dimensions stored as ``(width, height)`` (recommended
140        convention). If None, inferred from the TOML ``sig_shape``. If your
141        TOML stores detector shape as ``(Hdet, Wdet)``, this class normalizes
142        it to ``(Wdet, Hdet)`` for consistency.
143
144    packets : numpy.ndarray
145        Structured array of dtype
146        :data:`~Reader4D.dtypes.TP3_ACQ_DATA_PACKET_DTYPE` with fields
147        ``('address', 'count', 'itot')`` and length ``nnz``.
148
149    descriptors : numpy.ndarray
150        Structured array of dtype
151        :data:`~Reader4D.dtypes.TP3_BIN_DESCRIPTOR_DTYPE` with fields
152        ``('offset', 'packet_count')`` and length ``n_frames``.
153
154    header : dict or None
155        Normalized metadata/statistics dictionary assembled from TOML and
156        derived CSR statistics. Only available when ``return_header=True``.
157
158    output_dir : str
159        Resolved output directory used internally for saving figures and other
160        derived artifacts. A subfolder may be chosen based on ``values_role``.
161
162    Notes
163    -----
164    - CSR arrays are loaded using ``numpy.memmap`` to avoid up-front copies.
165    - Packet addresses are assumed to be linear indices into the detector plane
166      (0..W*H-1) unless your acquisition format specifies otherwise.
167    """

314                cmap=cmap,
315            )

318                plt.imsave(

321                    cmap=cmap,

324                plt.imsave(

327                    cmap=cmap,

337    def CSRLoader(
338        self,
339        indptr="indptr.raw",
340        indices="indices.raw",
341        data="values.raw",
342        values_role="count",
343        progress=True,
344        prefetch_mb=256,
345        return_header=False,
346    ):
347        """
348        Load a CSR triplet from ``self.in_dir`` using TOML metadata.
349
350        This method reads a TOML sidecar (``*.toml``) from ``self.in_dir`` to
351        determine file names, dtypes, and shapes. The CSR triplet is memory-
352        mapped (no up-front copy) and converted into:
353
354        - ``descriptors``: structured array with dtype
355          :data:`~Reader4D.dtypes.TP3_BIN_DESCRIPTOR_DTYPE` and fields
356          ``('offset', 'packet_count')`` (one row per frame/scan pixel)
357        - ``packets``: structured array with dtype
358          :data:`~Reader4D.dtypes.TP3_ACQ_DATA_PACKET_DTYPE` and fields
359          ``('address', 'count', 'itot')`` (one row per nonzero/packet)
360
361        Parameters
362        ----------
363        indptr : str, optional
364            Default filename for the CSR row-pointer file if not specified in
365            TOML (default ``"indptr.raw"``).
366
367        indices : str, optional
368            Default filename for the CSR indices file if not specified in TOML
369            (default ``"indices.raw"``).
370
371        data : str, optional
372            Default filename for the CSR values file if not specified in TOML
373            (default ``"values.raw"``).
374
375        values_role : {"count", "itot"}, optional
376            How to map CSR values into the packet fields. If ``"count"``,
377            values populate ``packets["count"]`` and ``packets["itot"]`` is set
378            to zero. If ``"itot"``, values populate ``packets["itot"]`` and
379            ``packets["count"]`` is set to zero.
380
381        progress : bool, optional
382            If True, prefetches memory maps using tqdm progress bars.
383
384        prefetch_mb : int, optional
385            Block size (MiB) used for sequential prefetching. Larger values
386            reduce Python overhead but increase I/O burst size.
387
388        return_header : bool, optional
389            If True, returns a normalized header/stats dictionary as a third
390            result.
391
392        Returns
393        -------
394        packets : numpy.ndarray
395            Structured array of shape ``(nnz,)`` with dtype
396            :data:`~Reader4D.dtypes.TP3_ACQ_DATA_PACKET_DTYPE`.
397
398        descriptors : numpy.ndarray
399            Structured array of shape ``(n_frames,)`` with dtype
400            :data:`~Reader4D.dtypes.TP3_BIN_DESCRIPTOR_DTYPE`.
401
402        header : dict or None
403            Normalized header/stats dictionary if ``return_header=True``,
404            otherwise None.
405
406        """
407        folder = self.in_dir
408
409        def _find_one(path, pattern):
410            """ Return the first filesystem entry matching a glob pattern 
411            in a directory. The results are sorted lexicographically to make
412            the selection deterministic."""
413            hits = sorted(glob.glob(os.path.join(path, pattern)))
414            return hits[0] if hits else None
415
416        if not os.path.isdir(folder):
417            raise FileNotFoundError(f"Folder not found: {folder}")
418
419        toml_file = _find_one(folder, "*.toml")
420        if not toml_file:
421            raise FileNotFoundError(
422                "No TOML file found in the input directory."
423                )
424
425        cfg = _read_toml(toml_file)
426        raw = cfg.get("raw_csr", {})
427        params = cfg.get("params", {})
428
429        # Resolve file names (TOML overrides defaults)
430        indptr_name = raw.get("indptr_file", indptr)
431        indices_name = raw.get("indices_file", indices)
432        data_name = raw.get("data_file", data)
433
434        # ---- Resolve dtypes
435        indptr_dtype = _DTYPE_MAP.get(
436            raw.get("indptr_dtype", "int64"), 
437            np.int64
438            )
439        indices_dtype = _DTYPE_MAP.get(
440            raw.get("indices_dtype", "int32"), 
441            np.int32
442            )
443        data_dtype = _DTYPE_MAP.get(
444            raw.get("data_dtype", "int32"), 
445            np.int32
446            )
447
448        nav_shape = params.get("nav_shape", None)
449        sig_shape = params.get("sig_shape", None)
450
451        indptr_path = os.path.join(folder, indptr_name)
452        indices_path = os.path.join(folder, indices_name)
453        data_path = os.path.join(folder, data_name)
454
455        for p in (indptr_path, indices_path, data_path):
456            if not os.path.exists(p):
457                raise FileNotFoundError(f"Missing CSR file: {p}")
458
459        # ---- Memory map (no copy)
460        indptr_mm = np.memmap(
461            indptr_path, 
462            mode="r", 
463            dtype=indptr_dtype
464            )
465        indices_mm = np.memmap(
466            indices_path, 
467            mode="r", 
468            dtype=indices_dtype
469            )
470        values_mm = np.memmap(
471            data_path, 
472            mode="r", 
473            dtype=data_dtype
474            )
475
476        if progress:
477            self._prefetch_memmap(
478                indptr_mm, 
479                "Prefetch indptr", 
480                prefetch_mb, 
481                position=0
482                )
483            self._prefetch_memmap(
484                indices_mm, 
485                "Prefetch indices", 
486                prefetch_mb, 
487                position=1
488                )
489            self._prefetch_memmap(
490                values_mm, 
491                "Prefetch values", 
492                prefetch_mb, 
493                position=2
494                )
495
496        # ---- CSR sanity
497        if indptr_mm.shape[0] < 2:
498            raise ValueError("indptr must have length >= 2 (n_frames + 1).")
499
500        nnz = int(indptr_mm[-1])
501        if nnz != indices_mm.size or nnz != values_mm.size:
502            raise ValueError(
503                f"CSR mismatch: indptr[-1]={nnz}, ",
504                " len(indices)={indices_mm.size}, len(values)={values_mm.size}"
505            )
506
507        n_frames = int(indptr_mm.shape[0] - 1)
508
509        # ---- Build descriptors
510        descriptors = np.empty(n_frames, dtype=BIN_DESCRIPTOR_DTYPE)
511        descriptors["offset"] = indptr_mm[:-1].astype(np.uint64, copy=False)
512
513        row_counts = (indptr_mm[1:]-indptr_mm[:-1]).astype(np.int64, copy=False)
514        
515        if np.any(row_counts < 0):
516            raise ValueError("indptr must be non-decreasing.")
517            
518        if np.any(row_counts > np.iinfo(np.uint32).max):
519            raise ValueError(
520                "A row has >2^32-1 packets; cannot fit into uint32.")
521
522        descriptors["packet_count"] = row_counts.astype(np.uint32, copy=False)
523
524        # ---- Build packets
525        packets = np.empty(nnz, dtype=ACQ_DATA_PACKET_DTYPE)
526        packets["address"] = indices_mm.astype(np.uint32, copy=False)
527
528        values_role = str(values_role).lower()
529        if values_role == "count":
530            packets["count"] = values_mm.astype(np.uint32, copy=False)
531            packets["itot"] = 0
532        elif values_role == "itot":
533            packets["itot"] = values_mm.astype(np.uint32, copy=False)
534            packets["count"] = 0
535        else:
536            raise ValueError("values_role must be 'count' or 'itot'")
537
538        # ---- Optional: validate against detector bounds (if provided)
539        if sig_shape:
540            hdet, wdet = int(sig_shape[0]), int(sig_shape[1])
541            det_size = hdet * wdet
542            if det_size > 0 and packets["address"].max(initial=0) >= det_size:
543                raise ValueError(
544                    f"Some packet addresses exceed detector size {sig_shape}: "
545                    f"max={int(packets['address'].max())}, limit={det_size-1}"
546                )
547
548        # ---- Optional: validate nav shape
549        if nav_shape:
550            h, w = int(nav_shape[0]), int(nav_shape[1])
551            if h * w != n_frames:
552                raise ValueError(
553                    f"nav_shape {nav_shape} implies {h*w} frames,",
554                    f" but indptr gives {n_frames} frames."
555                )
556
557        header = self._normalize_header(cfg, indptr_mm, indices_mm, values_mm)
558        return (packets, descriptors, header) if return_header else (packets, descriptors, None)

562    def _prefetch_memmap(self,
563            mm: np.memmap, desc: str, block_mb: int = 256, position: int = 0):
564        """
565        Sequentially prefetch a memory-mapped array into the OS page cache.
566    
567        The function walks the memmap in large contiguous blocks and performs a
568        temporary copy of each slice to *force* disk I/O. This warms the cache 
569        so that subsequent random access is much faster. No data is modified 
570        and the temporary slices are immediately discarded.
571    
572        Parameters
573        ----------
574        mm : np.memmap
575            Memory-mapped NumPy array to prefetch. Assumed 1-D; for multi-dim 
576            arrays, the first axis is iterated. Works with structured dtypes.
577        desc : str
578            Label shown next to the tqdm progress bar.
579        block_mb : int, optional
580            Target block size in megabytes for each read chunk (converted to a 
581            count of elements using ``mm.dtype.itemsize``). Larger values 
582            reduce Python overhead but increase each I/O burst. Default is 256.
583        position : int, optional
584            Row index for the tqdm bar (useful when displaying multiple bars 
585            at once). Default is 0.
586    
587        Returns
588        -------
589        None
590            The function has side effects only (it populates the OS cache).
591        """
592        itemsize = mm.dtype.itemsize
593        total = int(mm.shape[0])
594        
595        # Convert the desired block size from megabytes to "number of elements"
596        # Each element may be multi-byte (mm.dtype.itemsize).
597        block_elems = max(1, (block_mb * 1024 * 1024) // itemsize)    
598        
599        # How many blocks we'll need to cover the entire array.
600        steps = (total + block_elems - 1) // block_elems
601        
602        # Iterate from 0 up to 'total' in steps of 'bsz'
603        for start in tqdm(range(0, total, block_elems), total=steps, desc=desc,
604                          unit="blk", position=position, 
605                          leave=False, dynamic_ncols=True):
606            end = min(start + block_elems, total)
607            
608            # Slice the memmap for this block and copy it to a temporary array.
609            # The copy() call forces the OS to actually read the bytes from 
610            # disk into the page cache (and briefly into Python memory for this 
611            # slice), # "warming" the cache so later random access is fast.
612            # We discard the temporary array immediately by not storing it.
613            _ = mm[start:end].copy()  # forces IO

616    def _normalize_header(self, raw_cfg, indptr, indices, values):
617        """
618        Build a friendly, self-contained header dict from the parsed CSR TOML
619        plus a few derived statistics from the loaded arrays.
620    
621        Parameters
622        ----------
623        raw_cfg : dict-like
624            Parsed TOML config (e.g. the result of tomli/tomllib.load). 
625            Expected keys:
626            - raw_cfg["params"]: may contain "filetype", "nav_shape", 
627              "sig_shape", and optionally "threshold", "bias", "dwell_time_ns", 
628              "values_role".
629            - raw_cfg["raw_csr"]: file/dtype fields for indptr/indices/data.
630            
631        indptr : np.ndarray or np.memmap, shape (n_frames+1,)
632            CSR row pointer; must be non-decreasing. The last element equals 
633            nnz.
634            
635        indices : np.ndarray or np.memmap, shape (nnz,)
636            CSR column indices (detector linear addresses).
637            
638        values : np.ndarray or np.memmap, shape (nnz,)
639            CSR values (counts or itot).
640    
641        Returns
642        -------
643        stats : dict
644            A flat dict suitable for pretty-printing/logging. Contains:
645            - "filetype", "nav_shape", "sig_shape"
646            - "n_frames", "nnz"
647            - file names & dtypes: "indptr_file", "indices_file", "data_file",
648              "indptr_dtype", "indices_dtype", "data_dtype"
649            - size info: "bytes_indptr", "bytes_indices", "bytes_values"
650            - row stats: "packets_per_frame_min/max/mean/median", 
651                         "nonempty_frames", "occupancy" (fraction of frames
652                                                         with 1 packet)
653            - optional passthroughs if present: "threshold", "bias",
654              "dwell_time_ns", "values_role"
655        """
656        params = dict(raw_cfg.get("params", {}))
657        rcfg   = dict(raw_cfg.get("raw_csr", {}))
658    
659        # Basic fields from TOML (fallbacks if missing)
660        nav_shape = tuple(params.get("nav_shape", (None, None)))
661        sig_shape = tuple(params.get("sig_shape", (None, None)))
662        filetype  = params.get("filetype", "raw_csr")
663    
664        # Derived sizes
665        n_frames = int(indptr.shape[0] - 1)
666        nnz      = int(indptr[-1])
667    
668        # Per-frame packet counts and simple stats
669        row_counts = np.diff(indptr).astype(np.int64, copy=False)
670        nonempty   = int((row_counts > 0).sum())
671        occupancy  = float(nonempty / n_frames) if n_frames > 0 else 0.0
672    
673        # Assemble the header dict
674        stats = {
675            "filetype": filetype,
676            "nav_shape": nav_shape,           # (H, W) scan
677            "sig_shape": sig_shape,           # (Hdet, Wdet)
678            "n_frames": n_frames,
679            "nnz": nnz,
680    
681            # File fields from TOML (with sensible defaults)
682            "indptr_file":  rcfg.get("indptr_file",  "indptr.raw"),
683            "indices_file": rcfg.get("indices_file", "indices.raw"),
684            "data_file":    rcfg.get("data_file",    "values.raw"),
685    
686            # Dtype strings (prefer TOML if present; otherwise from arrays)
687            "indptr_dtype":  rcfg.get("indptr_dtype",  
688                                      str(
689                                          getattr(
690                                              indptr,  "dtype", "unknown"))),
691            "indices_dtype": rcfg.get("indices_dtype",
692                                      str(
693                                          getattr(
694                                              indices, "dtype", "unknown"))),
695            "data_dtype":    rcfg.get("data_dtype",    
696                                      str(
697                                          getattr(
698                                              values,  "dtype", "unknown"))),
699    
700            # Byte sizes
701            "bytes_indptr":  int(getattr(indptr,  "nbytes", 0)),
702            "bytes_indices": int(getattr(indices, "nbytes", 0)),
703            "bytes_values":  int(getattr(values,  "nbytes", 0)),
704    
705            # Row-count stats
706            "packets_per_frame_min":   int(row_counts.min(initial=0)),
707            "packets_per_frame_max":   int(row_counts.max(initial=0)),
708            "packets_per_frame_mean":  float(row_counts.mean() \
709                                             if row_counts.size else 0.0),
710            "packets_per_frame_median":float(np.median(row_counts) \
711                                             if row_counts.size else 0.0),
712            "nonempty_frames": nonempty,
713            "occupancy":       occupancy,
714        }
715    
716        # Optional passthroughs (only if present in TOML)
717        for opt in ("threshold", "bias", "dwell_time_ns", "values_role"):
718            if opt in params:
719                stats[opt] = params[opt]
720    
721        return stats

776    def get_diffractogram(self, 
777                      packets, descriptors, pattern_index,
778                      scan_dims=(1024, 768),
779                      detector_dims=(256, 256),
780                      dtype=np.uint32):
781        """
782        Recovers the 2D diffraction pattern for a single pixel of the scan 
783        image.
784    
785        Parameters
786        ----------
787        packets : numpy.ndarray
788            The raw packet data from the .advb file.
789            
790        descriptors : numpy.ndarray
791            The descriptor data from the .advb.desc file.
792       
793        pattern_index : integer
794            Index of the diffractogram to be reconstructed.
795        
796        scan_dims : tuple, optional
797            The (width, height) of the overall scan grid.
798        
799        detector_dims : tuple, optional
800            The (width, height) of the detector.
801    
802        Returns
803        -------
804        numpy.ndarray
805            A 2D array (256x256) representing the diffraction pattern.
806            
807        """
808        # sanity on dims
809        if (not isinstance(scan_dims, (tuple, list))) or len(scan_dims) != 2:
810            raise TypeError(
811                f"scan_dims must be (width,height), got {scan_dims}")
812            
813        if (not isinstance(detector_dims, (tuple, list))) or \
814            len(detector_dims) != 2:
815            raise TypeError(
816                f"detector_dims must be (width,height), got {detector_dims}")
817            
818        if not isinstance(pattern_index, int):
819            raise TypeError("pattern_index must be an integer.")
820    
821        det_width, det_height = map(int, detector_dims)   # (Wdet, Hdet)
822        n_pkts = int(packets.shape[0])
823    
824        # normalize descriptors to 1D arrays pc (packet_count) and off (offset) 
825        desc = np.asarray(descriptors)
826        if "packet_count" not in getattr(desc.dtype, "names", ()):
827            raise TypeError("descriptors must have field 'packet_count'.")
828    
829        pc = np.asarray(desc["packet_count"], dtype=np.int64).reshape(-1)
830        n_frames = pc.size
831        if not (0 <= pattern_index < n_frames):
832            raise IndexError("pattern_index out of bounds.")
833    
834        # prefer provided offset if it matches packets length;otherwise rebuild
835        use_provided_off = ("offset" in desc.dtype.names)
836        if use_provided_off:
837            off = np.asarray(desc["offset"], dtype=np.int64).reshape(-1)
838            # check consistency only on the last frame to avoid O(N) sums
839            if off[-1] + pc[-1] != n_pkts:
840                use_provided_off = False
841    
842        if not use_provided_off:
843            off = np.empty_like(pc, dtype=np.int64)
844            if pc.size:
845                off[0] = 0
846                if pc.size > 1:
847                    np.cumsum(pc[:-1], out=off[1:])
848    
849        # slice this frame
850        s = int(off[pattern_index])
851        e = s + int(pc[pattern_index])
852        if s < 0 or e < s or e > n_pkts:
853            # out of range  treat as empty
854            return np.zeros((det_height, det_width), dtype=dtype), 0
855    
856        frame_pkts = packets[s:e]
857    
858        # reconstruct diffractogram
859        # NOTE: image shape must be (Hdet, Wdet) for (y,x) indexing
860        img = np.zeros((det_height, det_width), dtype=dtype)
861        if frame_pkts.size:
862            addr = frame_pkts["address"].astype(np.int64, copy=False)
863            vals = frame_pkts[
864                getattr(
865                    self, "values_role", "count")].astype(np.uint64, 
866                                                          copy=False)
867    
868            det_size = det_width * det_height
869            valid = (addr >= 0) & (addr < det_size)
870            if not np.all(valid):
871                addr = addr[valid]; vals = vals[valid]
872    
873            # accumulate into raveled image for speed, then done
874            np.add.at(img.ravel(), addr, vals)
875    
876        count_sum = int(img.sum())
877        return img, count_sum

880    def get_count_itot(self,
881                       packets, 
882                       descriptors, 
883                       scan_dims, 
884                       show=True, 
885                       cmap="gray"):
886        """
887        Constructs 2D images of total counts and iToT from raw data and 
888        descriptors.
889    
890        Parameters
891        ----------
892        packets : numpy.ndarray
893            The raw packet data from the .advb file.
894        descriptors : numpy.ndarray
895            The descriptor data from the .advb.desc file.
896        scan_dims : tuple (width, height) 
897            Scan size specifications (dimensions of the micrograph)
898        show : boolean
899            Display reconstructed images. Default is True.
900        cmap : string
901            Colormap of the displayed images, used when show=True.
902            Default is gray.
903    
904        Returns
905        --------
906        count_image : numpy.ndarray 
907            Image of total event counts per pixel.
908        itot_image : numpy.ndarray 
909            Image of total iToT per pixel.
910        """
911        # Initialize dimensions
912        width, height = scan_dims
913        
914        # Initialize arrays for inserting values from packets
915        count_image = np.zeros((height, width), dtype=np.uint32)
916        itot_image = np.zeros((height, width), dtype=np.uint32)
917    
918        # Initialize counters
919        current_offset = 0
920        frame_idx = 0
921        
922        # Iterate through descriptors and packets
923        for descriptor in descriptors:
924            # Find the starting point (offset in descriptors)
925            frame_start = current_offset
926            
927            # Find the teminating point (offset + packet counts)
928            frame_end = frame_start + descriptor["packet_count"]
929            
930            # Extract all packets for the one pixel
931            frame_packets = packets[frame_start:frame_end]
932            
933            # Find the next starting point
934            current_offset += descriptor["packet_count"]
935            
936            # Get the coordinates of the pixel position within the array
937            frame_pos_x = frame_idx % width
938            frame_pos_y = frame_idx // width
939    
940            if frame_pos_y < height:
941                # Hit count image
942                count_image[frame_pos_y, frame_pos_x] = \
943                    np.sum(frame_packets["count"])
944                
945                # iToT image
946                itot_image[frame_pos_y, frame_pos_x] = \
947                    np.sum(frame_packets["itot"])
948            
949            # Move to the next frame
950            frame_idx += 1
951        
952        # Optionally display reconstructed images 
953        if show:
954            if np.sum(count_image) != 0:
955                r4dVisu.show_micrograph(
956                    count_image,
957                    title="Hit Counts Micrograph",
958                    cmap=cmap,
959                    save=False,
960                    filename=None,
961                    output_dir=self.out_dir,
962                    show=self.show
963                    )
964            else:
965                print("[INFO] Hit Counts not available.")
966                    
967            if np.sum(itot_image) != 0:
968                r4dVisu.show_micrograph(
969                    itot_image,
970                    title="iToT Micrograph",
971                    cmap=cmap,
972                    save=False,
973                    filename=None,
974                    output_dir=self.out_dir,
975                    show=self.show
976                    )
977            else:
978                print("[INFO] iToTs not available.")
979                
980                
981        if np.sum(count_image) == 0:
982            return None, itot_image
983        
984        elif np.sum(itot_image) == 0:
985            return count_image, None
986        
987        else:
988            return count_image, itot_image

338    def CSRLoader(
339        self,
340        indptr="indptr.raw",
341        indices="indices.raw",
342        data="values.raw",
343        values_role="count",
344        progress=True,
345        prefetch_mb=256,
346        return_header=False,
347    ):
348        """
349        Load a CSR triplet from ``self.in_dir`` using TOML metadata.
350
351        This method reads a TOML sidecar (``*.toml``) from ``self.in_dir`` to
352        determine file names, dtypes, and shapes. The CSR triplet is memory-
353        mapped (no up-front copy) and converted into:
354
355        - ``descriptors``: structured array with dtype
356          :data:`~Reader4D.dtypes.TP3_BIN_DESCRIPTOR_DTYPE` and fields
357          ``('offset', 'packet_count')`` (one row per frame/scan pixel)
358        - ``packets``: structured array with dtype
359          :data:`~Reader4D.dtypes.TP3_ACQ_DATA_PACKET_DTYPE` and fields
360          ``('address', 'count', 'itot')`` (one row per nonzero/packet)
361
362        Parameters
363        ----------
364        indptr : str, optional
365            Default filename for the CSR row-pointer file if not specified in
366            TOML (default ``"indptr.raw"``).
367
368        indices : str, optional
369            Default filename for the CSR indices file if not specified in TOML
370            (default ``"indices.raw"``).
371
372        data : str, optional
373            Default filename for the CSR values file if not specified in TOML
374            (default ``"values.raw"``).
375
376        values_role : {"count", "itot"}, optional
377            How to map CSR values into the packet fields. If ``"count"``,
378            values populate ``packets["count"]`` and ``packets["itot"]`` is set
379            to zero. If ``"itot"``, values populate ``packets["itot"]`` and
380            ``packets["count"]`` is set to zero.
381
382        progress : bool, optional
383            If True, prefetches memory maps using tqdm progress bars.
384
385        prefetch_mb : int, optional
386            Block size (MiB) used for sequential prefetching. Larger values
387            reduce Python overhead but increase I/O burst size.
388
389        return_header : bool, optional
390            If True, returns a normalized header/stats dictionary as a third
391            result.
392
393        Returns
394        -------
395        packets : numpy.ndarray
396            Structured array of shape ``(nnz,)`` with dtype
397            :data:`~Reader4D.dtypes.TP3_ACQ_DATA_PACKET_DTYPE`.
398
399        descriptors : numpy.ndarray
400            Structured array of shape ``(n_frames,)`` with dtype
401            :data:`~Reader4D.dtypes.TP3_BIN_DESCRIPTOR_DTYPE`.
402
403        header : dict or None
404            Normalized header/stats dictionary if ``return_header=True``,
405            otherwise None.
406
407        """
408        folder = self.in_dir
409
410        def _find_one(path, pattern):
411            """ Return the first filesystem entry matching a glob pattern 
412            in a directory. The results are sorted lexicographically to make
413            the selection deterministic."""
414            hits = sorted(glob.glob(os.path.join(path, pattern)))
415            return hits[0] if hits else None
416
417        if not os.path.isdir(folder):
418            raise FileNotFoundError(f"Folder not found: {folder}")
419
420        toml_file = _find_one(folder, "*.toml")
421        if not toml_file:
422            raise FileNotFoundError(
423                "No TOML file found in the input directory."
424                )
425
426        cfg = _read_toml(toml_file)
427        raw = cfg.get("raw_csr", {})
428        params = cfg.get("params", {})
429
430        # Resolve file names (TOML overrides defaults)
431        indptr_name = raw.get("indptr_file", indptr)
432        indices_name = raw.get("indices_file", indices)
433        data_name = raw.get("data_file", data)
434
435        # ---- Resolve dtypes
436        indptr_dtype = _DTYPE_MAP.get(
437            raw.get("indptr_dtype", "int64"), 
438            np.int64
439            )
440        indices_dtype = _DTYPE_MAP.get(
441            raw.get("indices_dtype", "int32"), 
442            np.int32
443            )
444        data_dtype = _DTYPE_MAP.get(
445            raw.get("data_dtype", "int32"), 
446            np.int32
447            )
448
449        nav_shape = params.get("nav_shape", None)
450        sig_shape = params.get("sig_shape", None)
451
452        indptr_path = os.path.join(folder, indptr_name)
453        indices_path = os.path.join(folder, indices_name)
454        data_path = os.path.join(folder, data_name)
455
456        for p in (indptr_path, indices_path, data_path):
457            if not os.path.exists(p):
458                raise FileNotFoundError(f"Missing CSR file: {p}")
459
460        # ---- Memory map (no copy)
461        indptr_mm = np.memmap(
462            indptr_path, 
463            mode="r", 
464            dtype=indptr_dtype
465            )
466        indices_mm = np.memmap(
467            indices_path, 
468            mode="r", 
469            dtype=indices_dtype
470            )
471        values_mm = np.memmap(
472            data_path, 
473            mode="r", 
474            dtype=data_dtype
475            )
476
477        if progress:
478            self._prefetch_memmap(
479                indptr_mm, 
480                "Prefetch indptr", 
481                prefetch_mb, 
482                position=0
483                )
484            self._prefetch_memmap(
485                indices_mm, 
486                "Prefetch indices", 
487                prefetch_mb, 
488                position=1
489                )
490            self._prefetch_memmap(
491                values_mm, 
492                "Prefetch values", 
493                prefetch_mb, 
494                position=2
495                )
496
497        # ---- CSR sanity
498        if indptr_mm.shape[0] < 2:
499            raise ValueError("indptr must have length >= 2 (n_frames + 1).")
500
501        nnz = int(indptr_mm[-1])
502        if nnz != indices_mm.size or nnz != values_mm.size:
503            raise ValueError(
504                f"CSR mismatch: indptr[-1]={nnz}, ",
505                " len(indices)={indices_mm.size}, len(values)={values_mm.size}"
506            )
507
508        n_frames = int(indptr_mm.shape[0] - 1)
509
510        # ---- Build descriptors
511        descriptors = np.empty(n_frames, dtype=BIN_DESCRIPTOR_DTYPE)
512        descriptors["offset"] = indptr_mm[:-1].astype(np.uint64, copy=False)
513
514        row_counts = (indptr_mm[1:]-indptr_mm[:-1]).astype(np.int64, copy=False)
515        
516        if np.any(row_counts < 0):
517            raise ValueError("indptr must be non-decreasing.")
518            
519        if np.any(row_counts > np.iinfo(np.uint32).max):
520            raise ValueError(
521                "A row has >2^32-1 packets; cannot fit into uint32.")
522
523        descriptors["packet_count"] = row_counts.astype(np.uint32, copy=False)
524
525        # ---- Build packets
526        packets = np.empty(nnz, dtype=ACQ_DATA_PACKET_DTYPE)
527        packets["address"] = indices_mm.astype(np.uint32, copy=False)
528
529        values_role = str(values_role).lower()
530        if values_role == "count":
531            packets["count"] = values_mm.astype(np.uint32, copy=False)
532            packets["itot"] = 0
533        elif values_role == "itot":
534            packets["itot"] = values_mm.astype(np.uint32, copy=False)
535            packets["count"] = 0
536        else:
537            raise ValueError("values_role must be 'count' or 'itot'")
538
539        # ---- Optional: validate against detector bounds (if provided)
540        if sig_shape:
541            hdet, wdet = int(sig_shape[0]), int(sig_shape[1])
542            det_size = hdet * wdet
543            if det_size > 0 and packets["address"].max(initial=0) >= det_size:
544                raise ValueError(
545                    f"Some packet addresses exceed detector size {sig_shape}: "
546                    f"max={int(packets['address'].max())}, limit={det_size-1}"
547                )
548
549        # ---- Optional: validate nav shape
550        if nav_shape:
551            h, w = int(nav_shape[0]), int(nav_shape[1])
552            if h * w != n_frames:
553                raise ValueError(
554                    f"nav_shape {nav_shape} implies {h*w} frames,",
555                    f" but indptr gives {n_frames} frames."
556                )
557
558        header = self._normalize_header(cfg, indptr_mm, indices_mm, values_mm)
559        return (packets, descriptors, header) if return_header else (packets, descriptors, None)

563    def _prefetch_memmap(self,
564            mm: np.memmap, desc: str, block_mb: int = 256, position: int = 0):
565        """
566        Sequentially prefetch a memory-mapped array into the OS page cache.
567    
568        The function walks the memmap in large contiguous blocks and performs a
569        temporary copy of each slice to *force* disk I/O. This warms the cache 
570        so that subsequent random access is much faster. No data is modified 
571        and the temporary slices are immediately discarded.
572    
573        Parameters
574        ----------
575        mm : np.memmap
576            Memory-mapped NumPy array to prefetch. Assumed 1-D; for multi-dim 
577            arrays, the first axis is iterated. Works with structured dtypes.
578        desc : str
579            Label shown next to the tqdm progress bar.
580        block_mb : int, optional
581            Target block size in megabytes for each read chunk (converted to a 
582            count of elements using ``mm.dtype.itemsize``). Larger values 
583            reduce Python overhead but increase each I/O burst. Default is 256.
584        position : int, optional
585            Row index for the tqdm bar (useful when displaying multiple bars 
586            at once). Default is 0.
587    
588        Returns
589        -------
590        None
591            The function has side effects only (it populates the OS cache).
592        """
593        itemsize = mm.dtype.itemsize
594        total = int(mm.shape[0])
595        
596        # Convert the desired block size from megabytes to "number of elements"
597        # Each element may be multi-byte (mm.dtype.itemsize).
598        block_elems = max(1, (block_mb * 1024 * 1024) // itemsize)    
599        
600        # How many blocks we'll need to cover the entire array.
601        steps = (total + block_elems - 1) // block_elems
602        
603        # Iterate from 0 up to 'total' in steps of 'bsz'
604        for start in tqdm(range(0, total, block_elems), total=steps, desc=desc,
605                          unit="blk", position=position, 
606                          leave=False, dynamic_ncols=True):
607            end = min(start + block_elems, total)
608            
609            # Slice the memmap for this block and copy it to a temporary array.
610            # The copy() call forces the OS to actually read the bytes from 
611            # disk into the page cache (and briefly into Python memory for this 
612            # slice), # "warming" the cache so later random access is fast.
613            # We discard the temporary array immediately by not storing it.
614            _ = mm[start:end].copy()  # forces IO
615

617    def _normalize_header(self, raw_cfg, indptr, indices, values):
618        """
619        Build a friendly, self-contained header dict from the parsed CSR TOML
620        plus a few derived statistics from the loaded arrays.
621    
622        Parameters
623        ----------
624        raw_cfg : dict-like
625            Parsed TOML config (e.g. the result of tomli/tomllib.load). 
626            Expected keys:
627            - raw_cfg["params"]: may contain "filetype", "nav_shape", 
628              "sig_shape", and optionally "threshold", "bias", "dwell_time_ns", 
629              "values_role".
630            - raw_cfg["raw_csr"]: file/dtype fields for indptr/indices/data.
631            
632        indptr : np.ndarray or np.memmap, shape (n_frames+1,)
633            CSR row pointer; must be non-decreasing. The last element equals 
634            nnz.
635            
636        indices : np.ndarray or np.memmap, shape (nnz,)
637            CSR column indices (detector linear addresses).
638            
639        values : np.ndarray or np.memmap, shape (nnz,)
640            CSR values (counts or itot).
641    
642        Returns
643        -------
644        stats : dict
645            A flat dict suitable for pretty-printing/logging. Contains:
646            - "filetype", "nav_shape", "sig_shape"
647            - "n_frames", "nnz"
648            - file names & dtypes: "indptr_file", "indices_file", "data_file",
649              "indptr_dtype", "indices_dtype", "data_dtype"
650            - size info: "bytes_indptr", "bytes_indices", "bytes_values"
651            - row stats: "packets_per_frame_min/max/mean/median", 
652                         "nonempty_frames", "occupancy" (fraction of frames
653                                                         with 1 packet)
654            - optional passthroughs if present: "threshold", "bias",
655              "dwell_time_ns", "values_role"
656        """
657        params = dict(raw_cfg.get("params", {}))
658        rcfg   = dict(raw_cfg.get("raw_csr", {}))
659    
660        # Basic fields from TOML (fallbacks if missing)
661        nav_shape = tuple(params.get("nav_shape", (None, None)))
662        sig_shape = tuple(params.get("sig_shape", (None, None)))
663        filetype  = params.get("filetype", "raw_csr")
664    
665        # Derived sizes
666        n_frames = int(indptr.shape[0] - 1)
667        nnz      = int(indptr[-1])
668    
669        # Per-frame packet counts and simple stats
670        row_counts = np.diff(indptr).astype(np.int64, copy=False)
671        nonempty   = int((row_counts > 0).sum())
672        occupancy  = float(nonempty / n_frames) if n_frames > 0 else 0.0
673    
674        # Assemble the header dict
675        stats = {
676            "filetype": filetype,
677            "nav_shape": nav_shape,           # (H, W) scan
678            "sig_shape": sig_shape,           # (Hdet, Wdet)
679            "n_frames": n_frames,
680            "nnz": nnz,
681    
682            # File fields from TOML (with sensible defaults)
683            "indptr_file":  rcfg.get("indptr_file",  "indptr.raw"),
684            "indices_file": rcfg.get("indices_file", "indices.raw"),
685            "data_file":    rcfg.get("data_file",    "values.raw"),
686    
687            # Dtype strings (prefer TOML if present; otherwise from arrays)
688            "indptr_dtype":  rcfg.get("indptr_dtype",  
689                                      str(
690                                          getattr(
691                                              indptr,  "dtype", "unknown"))),
692            "indices_dtype": rcfg.get("indices_dtype",
693                                      str(
694                                          getattr(
695                                              indices, "dtype", "unknown"))),
696            "data_dtype":    rcfg.get("data_dtype",    
697                                      str(
698                                          getattr(
699                                              values,  "dtype", "unknown"))),
700    
701            # Byte sizes
702            "bytes_indptr":  int(getattr(indptr,  "nbytes", 0)),
703            "bytes_indices": int(getattr(indices, "nbytes", 0)),
704            "bytes_values":  int(getattr(values,  "nbytes", 0)),
705    
706            # Row-count stats
707            "packets_per_frame_min":   int(row_counts.min(initial=0)),
708            "packets_per_frame_max":   int(row_counts.max(initial=0)),
709            "packets_per_frame_mean":  float(row_counts.mean() \
710                                             if row_counts.size else 0.0),
711            "packets_per_frame_median":float(np.median(row_counts) \
712                                             if row_counts.size else 0.0),
713            "nonempty_frames": nonempty,
714            "occupancy":       occupancy,
715        }
716    
717        # Optional passthroughs (only if present in TOML)
718        for opt in ("threshold", "bias", "dwell_time_ns", "values_role"):
719            if opt in params:
720                stats[opt] = params[opt]
721    
722        return stats

777    def get_diffractogram(self, 
778                      packets, descriptors, pattern_index,
779                      scan_dims=(1024, 768),
780                      detector_dims=(256, 256),
781                      dtype=np.uint32):
782        """
783        Recovers the 2D diffraction pattern for a single pixel of the scan 
784        image.
785    
786        Parameters
787        ----------
788        packets : numpy.ndarray
789            The raw packet data from the .advb file.
790            
791        descriptors : numpy.ndarray
792            The descriptor data from the .advb.desc file.
793       
794        pattern_index : integer
795            Index of the diffractogram to be reconstructed.
796        
797        scan_dims : tuple, optional
798            The (width, height) of the overall scan grid.
799        
800        detector_dims : tuple, optional
801            The (width, height) of the detector.
802    
803        Returns
804        -------
805        numpy.ndarray
806            A 2D array (256x256) representing the diffraction pattern.
807            
808        """
809        # sanity on dims
810        if (not isinstance(scan_dims, (tuple, list))) or len(scan_dims) != 2:
811            raise TypeError(
812                f"scan_dims must be (width,height), got {scan_dims}")
813            
814        if (not isinstance(detector_dims, (tuple, list))) or \
815            len(detector_dims) != 2:
816            raise TypeError(
817                f"detector_dims must be (width,height), got {detector_dims}")
818            
819        if not isinstance(pattern_index, int):
820            raise TypeError("pattern_index must be an integer.")
821    
822        det_width, det_height = map(int, detector_dims)   # (Wdet, Hdet)
823        n_pkts = int(packets.shape[0])
824    
825        # normalize descriptors to 1D arrays pc (packet_count) and off (offset) 
826        desc = np.asarray(descriptors)
827        if "packet_count" not in getattr(desc.dtype, "names", ()):
828            raise TypeError("descriptors must have field 'packet_count'.")
829    
830        pc = np.asarray(desc["packet_count"], dtype=np.int64).reshape(-1)
831        n_frames = pc.size
832        if not (0 <= pattern_index < n_frames):
833            raise IndexError("pattern_index out of bounds.")
834    
835        # prefer provided offset if it matches packets length;otherwise rebuild
836        use_provided_off = ("offset" in desc.dtype.names)
837        if use_provided_off:
838            off = np.asarray(desc["offset"], dtype=np.int64).reshape(-1)
839            # check consistency only on the last frame to avoid O(N) sums
840            if off[-1] + pc[-1] != n_pkts:
841                use_provided_off = False
842    
843        if not use_provided_off:
844            off = np.empty_like(pc, dtype=np.int64)
845            if pc.size:
846                off[0] = 0
847                if pc.size > 1:
848                    np.cumsum(pc[:-1], out=off[1:])
849    
850        # slice this frame
851        s = int(off[pattern_index])
852        e = s + int(pc[pattern_index])
853        if s < 0 or e < s or e > n_pkts:
854            # out of range  treat as empty
855            return np.zeros((det_height, det_width), dtype=dtype), 0
856    
857        frame_pkts = packets[s:e]
858    
859        # reconstruct diffractogram
860        # NOTE: image shape must be (Hdet, Wdet) for (y,x) indexing
861        img = np.zeros((det_height, det_width), dtype=dtype)
862        if frame_pkts.size:
863            addr = frame_pkts["address"].astype(np.int64, copy=False)
864            vals = frame_pkts[
865                getattr(
866                    self, "values_role", "count")].astype(np.uint64, 
867                                                          copy=False)
868    
869            det_size = det_width * det_height
870            valid = (addr >= 0) & (addr < det_size)
871            if not np.all(valid):
872                addr = addr[valid]; vals = vals[valid]
873    
874            # accumulate into raveled image for speed, then done
875            np.add.at(img.ravel(), addr, vals)
876    
877        count_sum = int(img.sum())
878        return img, count_sum

881    def get_count_itot(self,
882                       packets, 
883                       descriptors, 
884                       scan_dims, 
885                       show=True, 
886                       cmap="gray"):
887        """
888        Constructs 2D images of total counts and iToT from raw data and 
889        descriptors.
890    
891        Parameters
892        ----------
893        packets : numpy.ndarray
894            The raw packet data from the .advb file.
895        descriptors : numpy.ndarray
896            The descriptor data from the .advb.desc file.
897        scan_dims : tuple (width, height) 
898            Scan size specifications (dimensions of the micrograph)
899        show : boolean
900            Display reconstructed images. Default is True.
901        cmap : string
902            Colormap of the displayed images, used when show=True.
903            Default is gray.
904    
905        Returns
906        --------
907        count_image : numpy.ndarray 
908            Image of total event counts per pixel.
909        itot_image : numpy.ndarray 
910            Image of total iToT per pixel.
911        """
912        # Initialize dimensions
913        width, height = scan_dims
914        
915        # Initialize arrays for inserting values from packets
916        count_image = np.zeros((height, width), dtype=np.uint32)
917        itot_image = np.zeros((height, width), dtype=np.uint32)
918    
919        # Initialize counters
920        current_offset = 0
921        frame_idx = 0
922        
923        # Iterate through descriptors and packets
924        for descriptor in descriptors:
925            # Find the starting point (offset in descriptors)
926            frame_start = current_offset
927            
928            # Find the teminating point (offset + packet counts)
929            frame_end = frame_start + descriptor["packet_count"]
930            
931            # Extract all packets for the one pixel
932            frame_packets = packets[frame_start:frame_end]
933            
934            # Find the next starting point

848                    np.cumsum(pc[:-1], out=off[1:])

852        e = s + int(pc[pattern_index])

854            # out of range  treat as empty
855            return np.zeros((det_height, det_width), dtype=dtype), 0
856

858    
859        # reconstruct diffractogram
860        # NOTE: image shape must be (Hdet, Wdet) for (y,x) indexing
861        img = np.zeros((det_height, det_width), dtype=dtype)
862        if frame_pkts.size:
863            addr = frame_pkts["address"].astype(np.int64, copy=False)
864            vals = frame_pkts[
865                getattr(

867                                                          copy=False)

871            if not np.all(valid):
872                addr = addr[valid]; vals = vals[valid]
873    
874            # accumulate into raveled image for speed, then done
875            np.add.at(img.ravel(), addr, vals)
876    
877        count_sum = int(img.sum())
